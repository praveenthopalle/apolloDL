About Us

Apollo

APOLLO stands for AI Program for Overviewing Legal Literature Online.

APOLLO program allows you to efficiently and accurately categorize patent and journal data by leveraging the powers of AI (machine learning algorithms). The program allows you to build supervised and unsupervised machine learning models for patent and journal classification and save the models for later use. You can re-use a previously trained model to quickly categorize new patent and journal data. Moreover, you can incrementally update the AI model any time by providing additional training data. You can quickly compare and rank patents to find possible infringement cases.

For a detailed description of each of the programs within APOLLO, Please refer to the User Manual
